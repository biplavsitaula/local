# Color Variables Reference

This document provides a reference for all semantic color variables used in the Flame Beverage website.

## Primary Colors (Orange/Flame Theme)
- `--color-primary-text` - Primary text color (orange)
- `--color-primary-btn` - Primary button background
- `--color-primary-hover` - Primary button hover state
- `--color-primary-border` - Primary border color
- `--color-primary-light` - Lighter shade of primary
- `--color-primary-dark` - Darker shade of primary

## Secondary Colors (Red)
- `--color-secondary-text` - Secondary text color (red)
- `--color-secondary-btn` - Secondary button background
- `--color-secondary-hover` - Secondary button hover state
- `--color-secondary-border` - Secondary border color
- `--color-secondary-light` - Lighter shade of secondary
- `--color-secondary-dark` - Darker shade of secondary

## Tertiary Colors (Yellow/Gold)
- `--color-tertiary-text` - Tertiary text color (golden)
- `--color-tertiary-btn` - Tertiary button background
- `--color-tertiary-hover` - Tertiary button hover state
- `--color-tertiary-border` - Tertiary border color
- `--color-tertiary-light` - Lighter shade of tertiary
- `--color-tertiary-dark` - Darker shade of tertiary

## Text Colors
- `--color-text-primary` - Main text color
- `--color-text-secondary` - Secondary/muted text
- `--color-text-tertiary` - Tertiary text
- `--color-text-inverse` - Text on colored backgrounds
- `--color-text-golden` - Golden accent text

## Background Colors
- `--color-bg-primary` - Main background
- `--color-bg-secondary` - Card/secondary background
- `--color-bg-tertiary` - Tertiary background
- `--color-bg-muted` - Muted background
- `--color-bg-overlay` - Overlay background

## Border Colors
- `--color-border-primary` - Main border
- `--color-border-secondary` - Secondary border
- `--color-border-muted` - Muted border
- `--color-border-primary-accent` - Accent border

## Button Colors
- `--color-btn-primary` - Primary button background
- `--color-btn-primary-hover` - Primary button hover
- `--color-btn-primary-text` - Primary button text
- `--color-btn-primary-border` - Primary button border
- `--color-btn-secondary` - Secondary button background
- `--color-btn-secondary-hover` - Secondary button hover
- `--color-btn-secondary-text` - Secondary button text
- `--color-btn-secondary-border` - Secondary button border
- `--color-btn-outline` - Outline button background
- `--color-btn-outline-hover` - Outline button hover
- `--color-btn-outline-text` - Outline button text
- `--color-btn-outline-border` - Outline button border

## Status Colors
- `--color-success-bg` - Success background
- `--color-success-text` - Success text
- `--color-error-bg` - Error background
- `--color-error-text` - Error text
- `--color-warning-bg` - Warning background
- `--color-warning-text` - Warning text
- `--color-info-bg` - Info background
- `--color-info-text` - Info text

## Usage Examples

### In Tailwind Classes
Use the semantic utility classes:
- `text-primary-text` - Primary text color
- `bg-primary-btn` - Primary button background
- `border-primary-border` - Primary border
- `btn-primary` - Complete primary button styling

### In CSS/Inline Styles
```css
color: var(--color-primary-text);
background-color: var(--color-primary-btn);
border-color: var(--color-primary-border);
```

### In Tailwind Arbitrary Values
```html
<div className="text-[var(--color-primary-text)]">
<div className="bg-[var(--color-primary-btn)]">
```
